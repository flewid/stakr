== Installation ==
1. Please install plugin "MainWP Dashboard" and active it before install Clone Extension plugin (get the MainWP Dashboard plugin from url:http://mainwp.com/)
2. Upload the `mainwp-branding-extension` folder to the `/wp-content/plugins/` directory
3. Activate the Child Branding Extension plugin through the 'Plugins' menu in WordPress

== Screenshots ==
1. Enable or Disable extension on the "Extensions" page in the dashboard
