== Installation ==
1. Please install plugin "MainWP Dashboard" and active it before install MainWP Clean and Lock Extension plugin (get the MainWP Dashboard plugin from url:http://mainwp.com/)
2. Upload the `mainwp-secure-and-clean-dashboard-extension` folder to the `/wp-content/plugins/` directory
3. Activate the MainWP Clean and Lock Extension plugin through the 'Plugins' menu in WordPress
